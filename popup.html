Tips

Gunakan link watch, shorts, atau youtu.be. File akan diunduh menggunakan penyedia pihak ketiga (piped.video). Anda dapat menggantinya lewat konfigurasi di bagian Pengaturan lanjutan.

Pengaturan lanjutan

Gunakan placeholder {videoId} sebagai lokasi ID video. Contoh: https://server-anda/api/streams/{videoId} atau http://localhost:3000/download?target={videoId}.
